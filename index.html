HTML文档的基本结构

HTML（HyperText Markup Language，超文本标记语言）是用于创建网页的标准标记语言。
它不是一种编程语言，而是一种标记语言（markup language），用于描述网页的结构和内容。
HTML文档是由HTML元素组成的，这些元素由标签（tags）、属性和元素内容等组成。

    HTML文档的基本结构通常包括以下几个部分：
  1. <!DOCTYPE html>：声明文档类型和版本。
  2. <html>：html标签是HTML页面的根元素。
  3. <head>：包含了所有的头部标签元素。如<title>（定义网页标题）、<style>（定义内部样式表）、<meta>（提供有关HTML文档的元信息）、<link>（定义文档与外部资源之间的关系）、<script>（定义客户端脚本，比如JavaScript）等。
  4. <body>：包含了所有的主体内容，如文本、图片、音频、视频、超链接、表格、列表等。

<div>标签

<div>标签是一个容器标签，主要用于将文档分割为独立的、不同的部分;
它本身没有特定的语义含义、没有默认的样式，通常与 CSS 结合使用来对页面布局进行划分和控制;
<div>标签可以嵌套其他<div>标签或任何块级元素;
例如在这个块中，div被设置为class="label_div"，所有包裹在其中的背景颜色都被设置为#ebebebdc

<span>标签

<span> 标签在 HTML 中是一个行内元素（inline element），
它经常被用来对文本中的一部分应用样式或者作为其他行内元素的容器。
<span> 标签本身并不表示任何特定的内容或格式，
而是通过与 CSS 样式或其他属性结合使用，来提供对文本或其他行内内容的样式控制。

  • <table>元素定义了表格。border属性用于给表格添加边框（在现代网页设计中通常使用CSS来定义表格的样式）
  • <thead>元素包含了表格的表头部分，虽然<thead>不是必需的，但它为表格的结构提供了语义上的意义
  • <tr>元素定义了表格的行
  • <th>元素定义了表头单元格，通常包含粗体居中的文本。默认文本居中，也可以使用CSS来修改
  • <tbody>元素包含了除表头以外的表格的主体部分。<tbody>也不是必需的，但同样为表格的结构提供语义意义
  • <td>元素定义了表格数据单元格，通常包含普通的文本

在HTML5中，<th>和<td>元素也可以包含其他HTML元素，比如链接、图片等。

行内元素

HTML中的行内元素（也称为内联元素、内嵌元素或直进式元素）
HTML5中的一些元素具有灵活的显示特性，可以通过CSS样式调整其显示方式，
比如将行内元素设置为块级元素或将块级元素设置为行内元素。
这通常是通过修改元素的display属性来实现的，
例如display: inline;或display: block;
常见行内元素主要包括以下这些:

    文本相关的元素：
  1. <a> ：超链接
  2. <b> ：粗体
  3. <i> ：斜体
  4. <u> ：下划线
  5. <strong> ：强调，通常为粗体
  6. <em> ：强调，通常为斜体
  7. <span> ：通用内联容器
  8. <br> ：换行符
    表单相关的元素：
  1. <input> ：输入字段
  2. <label> ：表单字段标签
  3. <button> ：按钮
  4. <textarea> ：多行文本输入框
  5. <select> ：下拉选择框
    图像和多媒体：
  1. <img> ：图片
  2. <video> ：视频
  3. <audio> ：音频
    其他：
  1. <sub> ：下标文本
  2. <sup> ：上标文本
  3. <code> ：计算机代码
  4. <var> ：定义变量
  5. <samp> ：定义程序样本输出
  6. <kbd> ：定义键盘输入
  7. <cite> ：引用文本
  8. <q> ：短引用

表单元素

  1. <form>标签

    <form> 元素用于创建HTML表单以收集用户输入。
    <form> 元素可以包含各种表单控件，如文本字段、复选框、单选按钮、提交按钮等。
    表单还可以包含<input>、<textarea>、<button>、<select> 和 <option>等元素。

  2. <input>标签

    <input> 元素用于创建输入字段，如文本字段、密码字段、单选按钮、复选框等。
    <input> 元素的 type 属性决定了输入字段的类型。

    例: 






    
<form>
    <label for="fname">名字:</label><br>
    <input type="text" id="fname" name="fname"><br>
    <label for="fpassword">密码:</label><br>
    <input type="password" id="fpassword" name="fpassword"><br>
    <input type="checkbox" id="vehicle1" name="vehicle1" value="Bike">
    <label for="vehicle1"> 我有一辆自行车</label><br>
    <input type="radio" id="genderMale" name="gender" value="male">
    <label for="genderMale">男</label><br>
</form>

  3. <textarea>标签

    <textarea> 元素用于创建多行文本字段，用户可以输入多行文本。

    例: 


    
<form>
    <label for="message">留言:</label><br>
    <textarea id="message" name="message" rows="4" cols="50"></textarea><br>
</form>

  4. <button>标签

    <button> 元素用于在表单中创建点击按钮。
    它可以被用来提交表单、重置表单内容或执行其他JavaScript代码。

    例: 
    
<form>
    <button type="submit">提交
    <button type="reset">重置
</form>

  5. <select>和<option>标签

    <select> 元素用于创建下拉列表，<option> 元素用于定义列表中的选项。

    例: 

    
<form>
    <label for="cars">选择一辆车:</label>
    <select id="cars" name="cars">
        <option value="volvo">Volvo</option>
        <option value="saab">Saab</option>
        <option value="mercedes">Mercedes</option>
        <option value="audi">Audi</option>
    </select>
</form>

  6. <label>标签

    <label> 元素为 <input> 元素定义标签（描述）。
    <label> 元素的 for 属性应与 <input> 元素的 id 属性匹配。

  7. form元素的 action 和 method 属性

    action 属性定义了表单数据提交时将发送到的URL。
    method 属性定义了用于发送表单数据的HTTP方法（通常是"get"或"post"）。

  8. 表单验证

    HTML5 引入了一些新的输入类型，如 email、url、number、date 等，
    它们可以帮助浏览器自动验证用户输入的数据。
    此外，还可以使用 required 属性来强制用户填写某些字段。
